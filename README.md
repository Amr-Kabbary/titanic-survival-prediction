# Titanic Survival Prediction (Beginner Machine Learning Project)

## Overview
This is one of my early machine learning projects completed as part of my learning journey on Kaggle.  
The goal was to build a predictive model that estimates passenger survival on the Titanic using available passenger data.

Although this was done at a beginner stage, it helped me understand the full end-to-end machine learning workflow, from data exploration to model submission.

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Workflow Summary
- Explored and analyzed the dataset
- Handled missing values and cleaned data
- Performed basic feature engineering
- Built a machine learning model using Random Forest Classifier
- Evaluated model performance
- Generated a Kaggle submission file

---

## Model Used
- Random Forest Classifier
